   1 ## Coding Standards
   2 *   **Types (Classes/Structs/Enums):** Use `camelCase` (e.g., `dungeonMap`).
   3 *   **Methods & Functions:** Use `camelCase` (e.g., `loadLevel()`).
   4 *   **Member Variables:** Use `m_camelCase` for private members (e.g., `m_currentFloor`) and `camelCase` for public members.
   5 *   **File Names:** Match the class name (e.g., `GameStateManager.cpp`).
